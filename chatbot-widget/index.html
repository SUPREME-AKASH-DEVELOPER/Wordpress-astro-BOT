Demski Group Chatbot Widget

Standalone embeddable lead-capture chat — click the avatar in the bottom-right corner to try it.

Embed snippet

Add this single line before </body> on any page (WordPress, Elementor, Astro, or plain HTML):

<script
  src="https://YOUR-DEPLOYMENT.vercel.app/loader.js"
  data-widget-src="https://YOUR-DEPLOYMENT.vercel.app/widget.js"
  async></script>

Optional configuration

Pass overrides as data-* attributes on the loader script tag:

data-avatar               // override avatar image URL
data-avatar-fallback      // fallback avatar URL
data-calendly             // Calendly scheduling URL
data-bot-name             // Display name (default "Erin")
data-bot-title            // Display title (default "The Demski Group")
